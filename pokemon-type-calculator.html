Pokémon Type Calculator

Calculate type effectiveness, damage multipliers, and weaknesses for Pokémon battles (Gen 9). Supports single and dual types.

About the Pokémon Type Calculator

This calculator helps you understand type matchups in Pokémon battles. Select the attacking move type and the defending Pokémon's type(s) to see how effective the move will be. It supports both single and dual-type Pokémon and uses the official Gen 9 type chart.

Effectiveness: Super Effective (2x), Not Very Effective (0.5x), No Effect (0x), Immune (0x). Dual types multiply the multipliers.

Type Effectiveness

How to Use the Pokémon Type Calculator

  1. Select the type of the attacking move from the dropdown
  2. Choose the first type of the defending Pokémon
  3. (Optional) Add a second type for dual-type Pokémon
  4. Click "Calculate Type Effectiveness" to see the result
  5. Use the color-coded output to understand battle advantages

Note: This calculator uses the official Gen 9 type chart. Dual types multiply the individual multipliers.

Pokémon Battle Tips

  • Super Effective: Always prioritize moves that deal 2x or 4x damage
  • Resistances: Avoid 0.5x moves; switch to a better matchup
  • Immunities: 0x moves are useless; change strategy immediately
  • Dual Types: Multipliers multiply (0.5x × 0.5x = 0.25x)
  • STAB: Same-Type Attack Bonus adds 1.5x to matching moves
